# Currency Converter (WinForms)

## Project Overview
This is a Windows-based desktop application developed to perform real-time currency conversions between various international and custom currencies. [cite_start]The tool provides a graphical user interface for easy interaction and precise mathematical calculations based on defined exchange rates[cite: 44, 45].

## Key Features
* [cite_start]**Multi-Currency Support**: Allows conversion between PLN, EUR, USD, and a custom currency VD.
* **Interactive Controls**:
    * [cite_start]**Manual Entry**: Users can type specific amounts directly into the input text box.
    * [cite_start]**Scroll Bar Support**: An integrated horizontal scroll bar allows for rapid adjustment of values.
* [cite_start]**Real-Time Logic**: The application automatically recalculates results whenever the input amount or currency selection changes.
* [cite_start]**Intuitive Interface**: Uses GroupBoxes and RadioButtons to clearly separate source and target currency selections.
